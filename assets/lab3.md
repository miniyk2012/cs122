# 实验三：内部模块化的命令行菜单小程序V2.0

## 实验步骤
1. 分别编写linklist.h, linklist.c, menu.c文件
   1. linklist.h: 定义数据结构和操作的声明
   2. linklist.c: 实现操作
   3. menu.c: 菜单功能
2. 编译: `gcc menu.c linklist.c -o menu`
3. 运行: `./menu`
